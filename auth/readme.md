# MyRIOI TMA API

> Backend API for the MyRIOI TMA Auth service

## Usage

Rename the " to ".env.example" and update the values/settings to your own

## Install Dependences
```
npm install
```
## Run App
```
# Run in devevelopment mode
npm run dev

# Run in Production mode
npm start
```
