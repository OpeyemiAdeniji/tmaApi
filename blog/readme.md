# MYRIOI TMA-API

> Backend API for the MYRIOI TMA Blog Service 

## Usage

Rename the ".env.example" file to .env and update the values/settings to your own

## Install Dependences
```
npm install --force
```
## Run App
```
# Run in devevelopment mode
npm run dev

# Run in Production mode
npm start
```
